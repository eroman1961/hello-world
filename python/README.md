**Hello World (Python)**
A tiny README describing the `helloworld.py` example.

- **File:** `helloworld.py` — defines a function `helloworld()` that returns the string "Hello, World!" and prints it when the script is run directly.
- **Functionality:** `helloworld()` returns `"Hello, World!"`.
- **Python version:** Works on Python 3.x.

Usage
- Run the script from the `python` folder:

```
python helloworld.py
```

Expected output
- The script prints:

```
Hello, World!
```

Example (importing the function)
- You can import and call the function from other Python code:

```python
from helloworld import helloworld
print(helloworld())  # -> Hello, World!
```

Notes
- This example is intentionally minimal — useful for testing Python setup or demonstrating a simple function with a `__main__` entrypoint.
